📋 Project Overview

DeepGuard AI is a state-of-the-art deepfake detection system that combines CNN for spatial feature extraction and LSTM for temporal pattern analysis.

  • Architecture: EfficientNet-B0 + Bi-LSTM with Attention
  • Dataset: DFD (Google/Jigsaw) - 3,431 videos
  • Framework: PyTorch 2.0 + Flask
  • Face Detection: MTCNN (20 frames per video)

🎯 Performance

Overall Accuracy 91.6%
Fake Detection Rate 95%
Real Detection Rate 63%

* Model prioritizes catching fakes (95%) over false positives
